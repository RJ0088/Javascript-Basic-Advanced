## Increment & Decrement Operator

Example one **Without inc/dec operator**

```js
let num = 1;
num = 1 + num;
// or
num += 1;
```

**With inc/dec** Substruction

```js
let num = 1;
num++; // num = 2
num++; // num = 3
num++; // num = 4
```

**Note**: We can only use `++` or `--` in a variable not any number. like `5++` or `3--`
